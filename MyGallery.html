üìù Cara Upload Foto Baru:

  1. Buka Repository GitHub
  2. Masuk ke folder images
  3. Klik Add file ‚Üí Upload files
  4. Drag & drop foto atau klik "choose your files"
  5. Nama file pakai format: kategori-nama.jpg (contoh: pantai-bali.jpg)

  8. Refresh halaman website untuk lihat foto baru!
